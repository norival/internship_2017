# -- notes ---------------------------------------------------------------------

Culture:
C: tout dans la même cellule quand multivariété
R: sur plusieurs lignes

Surface:
C: surface totale de la parcelle
R: surface proportionnelle de chaque culture quand plusieurs cultures

Année + année enquête :
R: que 2011 ?

Divers:
R: Code pour le matériel

Irrigation:
Pour C|Rem_irrigation, voir R|irrig, il y a parfois des remarques

Travail du sol:
C: séparés dans pusieurs cellules
R: Plusieurs trucs dans la même cellule + colonne remarques
R: Ils sont séparés dans travail_sol__i [1-7]

Produits + semis:
R: tout est regroupé dans intervention
  R|intervention|activateur
  R|intervention|adjuvant
  R|intervention|binage

# -- algorithme BDD_Robin vers BDD_culture -------------------------------------

POUR chaque niveau de R$IDparcelle:
  Créer un tableau vide avec les noms de colonnes de C et beaucoup de lignes
  Récupérer les infos uniques disponibles dans la première ligne:
    ID_Parcelle
    ID_Exploitation
    Année_SuiviParcelle = 2011
    point_X
    point_Y
    Type_Culture
    Type_CultureSimplifiée
    Surface_ha
    Type_Exploitation
    Type_de_sol
    Interculture
    Mélange
    Date_Semis
    Densité_Semis
    Unité_DensitéSemis = kg/ha
    ObjRdt (qtx)
    Rdt_Qtx
    Culture_précédente
    Culture_suivante

  # Récupérer les informations mutliples, 1 info par ligne
  POUR chaque ligne de R$intervention:
    SI fertilisation:
      C|Produit_Ferti <- R|produit [R|intervention == fertilisation]
      C|Dose_Ferti <- R|dose passage [R|intervention == fertilisation]
        "dose.passage"
      C|Unité_dose <- R|unité/ha [R|interventaion == fertilisation]
        "unité.ha"
      C|Date_Ferti <- R|date.V0 [R|intervention == fertilisation]

    SI %in% [fongicide,herbicide,insecticide,molluscide,raticide,répulsif]:
      C|Protection_visée <- R|intervention
      C|Produit_phyto <- R|produit
      C|Dose_Phyto <- R|dose passage
        "dose.passage"
      C|Unité_dose <- R|unité/ha
        "unité.ha"
      C|Date_Phyto <- R|date.V0

  POUR chaque colonnes dans travail_sol__[1-7]:
    SI cellule != "0":
      C|Type_Tsol <- R|travail_sol__i

  Remplacer les NA par des "" pour que le format soit semblable à C
